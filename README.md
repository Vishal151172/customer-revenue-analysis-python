# Customer Repeat Purchase & Revenue Behavior Analysis (Python)

## Project Overview
This project analyzes retail transaction data to evaluate repeat customer behavior and revenue contribution using Python and Pandas.

## Tools & Libraries Used
- Python
- Pandas
- Matplotlib

## Key Analysis Performed
- Total revenue calculation
- Customer purchase frequency analysis
- Repeat vs New customer segmentation
- Revenue contribution comparison
- Average Order Value (AOV) analysis
- Visualization of customer value comparison

## Key Insights
- Repeat customers generated approximately 94% of total revenue.
- Repeat customers had higher average order value compared to new customers.
- Strong customer retention impact on overall revenue.

## Business Value
The analysis highlights the importance of customer retention strategies for revenue growth and business sustainability.
